# 🌲 SABD2026 — Machine Learning Project

## 📌 Описание проекта

Проект выполнен в рамках курса SABD2026. Цель — построить модель машинного обучения для предсказания спектров на основе нормализованных входных данных.

## 🎯 Задача

На основе 10 входных признаков предсказать 60 целевых переменных (спектр ISO Cf-252).

**Тип задачи:** Многорежимная регрессия (Multi-output Regression)

## 📊 Датасет

- **Файл:** `normalized_inputs_and_targets_gsf.csv`
- **Размер:** 500 713 строк × 70 столбцов
- **Признаки (X):** 10 нормализованных входных параметров
- **Целевые переменные (y):** 60 значений энергии спектра

## 🛠️ Технологии

| Библиотека | Версия | Назначение |
|------------|--------|------------|
| Python | ≥ 3.13 | Язык программирования |
| pandas | ≥ 3.0.1 | Работа с данными |
| numpy | ≥ 2.4.3 | Математические операции |
| scikit-learn | ≥ 1.8.0 | Модель машинного обучения |
| matplotlib | ≥ 3.10.8 | Визуализация |

## 🌲 Модель

Используется **RandomForestRegressor** (случайный лес для регрессии).

### Параметры модели:
- Количество деревьев: 50
- Максимальная глубина: 15
- Минимальное число образцов для разделения: 10
- Минимальное число образцов в листе: 5
